# 📝 To-Do List Web App

Live Demo: [To-Do List](https://muhammad-ilham21.github.io/To-do-List/)

---

## 🌟 Overview
This is a simple and interactive **To-Do List application** built with **HTML, CSS, and JavaScript**.  
It helps users organize tasks with due dates, manage them easily, and keep track of progress using persistent storage.

---

## 🚀 Features
- ➕ **Add Tasks** – Enter a task name and due date.  
- 🗑️ **Delete Tasks** – Remove tasks instantly with a delete button.  
- 💾 **Persistent Storage** – Tasks are saved in the browser using `localStorage`, so they remain even after refreshing or closing the page.  
- ⌨️ **Keyboard Support** – Press **Enter** to quickly add a task.  
- 🎨 **Clean Layout** – Grid-based design for neat task display.  

---

## 🛠️ Technologies Used
- **HTML** – Structure of the app  
- **CSS** – Styling with grid layout and buttons  
- **JavaScript** – Logic for adding, deleting, and saving tasks
